# /tbr list 

View your own or another user’s TBR (To Be Read) list. Use this command to see what books you (or someone else) want to read, have finished, or did not finish—along with each book’s current status.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/tbr list [user]
```
- `[user]` – (Optional) Mention another user to view their TBR list. If omitted, shows your own list.

:::warning
**If you want some ideas you can look at other peoples lists for inspiration!**
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/tbr list
```
> This will display your current TBR list, showing each book’s title, author, and status (tbr, finished, or dnf)
```bash
/tbr list @Jordan
```
> This will show @Jordan’s TBR list, including their books and statuses

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Book status makes it easy to track your reading progress (tbr = want to read, finished = read, dnf = did not finish)
> - Viewing someone else’s TBR list is great for finding reading buddies or getting book ideas

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/tbr add` – Add a book to your TBR list

`/tbr remove` – Remove a book from your TBR
