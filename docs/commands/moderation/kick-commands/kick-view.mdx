# /kick view

View detailed information about a specific kick case logged by Pandoryx. This command is part of the Moderation module.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/kick view <case>
```
- `<case>` – The case number of the kick you want to view. This number is listed in `/kick list.`

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/kick view 2
```
This will:
- Retrieve the full details for kick case #2
- Show information such as:
- Kicked user's ID
- Moderator who issued the kick
- Date and time of the action
- Reason for the kick

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Use `/kick list` to find the case number if you don’t have it on hand.
> - If the case number doesn’t exist, the bot will respond with an error.
> - Helpful for reviewing past moderation actions or keeping a paper trail.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/kick add` – Kick a user and log the action

`/kick list` – View all logged kick cases

`/kick remove` – Remove a kick case from the logs
