# /ban view

View detailed information about a specific ban case that has been logged by Pandoryx.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/ban view <case>
```
- `<case>` – The case number as listed in `/ban list`. This is not the user ID — it's the index tied to a specific logged ban.

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/ban view 4
```
This will:
- Retrieve and display the full details for ban case #4
- Include information such as:
- Banned user's ID
- Moderator who issued the ban
- Date/time of the action
- Reason given

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - If the provided case number doesn’t exist, the bot will respond with an error message.
> - Use `/ban list` to see available case numbers and choose the one you want to inspect.
> - Useful for reviewing past moderation decisions or keeping internal transparency.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/ban add` – Ban a user and create a case log

`/ban list` – View all logged ban cases

`/ban remove` – Unban a user and delete the case
