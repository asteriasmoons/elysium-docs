# /ban remove

Unban a user from the server and remove their ban case from Pandoryx’s moderation logs.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/ban remove <userid>
```
- `<userid>` – The Discord user ID of the person you want to unban and remove from the ban logs.

:::success
This will immediately unban the user and delete their associated ban case from Pandoryx's records.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/ban remove 123456789012345678
```
This will:
- Unban the user with ID 123456789012345678
- Remove the corresponding moderation case from your server’s ban log
- Return a confirmation once the action is complete

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - You must use the user ID, even if the user is no longer in the server.
> - The command will fail if the user is not currently banned.
> - If you want to preserve the ban case but still unban the user, use Discord’s native unban option instead.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/ban add` – Ban a user and create a log entry

`/ban list` – View all recorded ban cases

`/ban view` – View a specific ban case by index
