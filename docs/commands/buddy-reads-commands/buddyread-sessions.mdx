# /buddyread sessions 

View all of your current active buddy reading sessions. This command helps you keep track of your ongoing buddy reads, see who you’re paired with, and manage your current reading partnerships at a glance.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/buddyread sessions
```

:::info
No options or inputs are required — just run the command to see your active sessions.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/buddyread sessions
```
This will:
- Display a list or embed of all your active buddy reading sessions
- Show details for each session, including the book title, your buddy’s username, and session start date
- Allow you to easily reference or manage your ongoing buddy reads

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Only sessions that are currently active (not finished or left) are shown
> - Use this command regularly to stay organized with multiple buddy reads
> - If you want to see past or finished sessions, check your message history or finished status elsewhere
> - Useful for quickly jumping to related actions like messaging, finishing, or leaving a session

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/buddyread status` – View the status of a specific buddy read session

`/buddyread dm` – Send a direct message to your buddy

`/buddyread finish` – Mark a session as finished
