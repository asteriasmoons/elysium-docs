# /buddyread announce

Announce you’re reading a specific book and looking for a reading buddy. Use this command to post a public call for someone to join you in reading a book together, optionally adding a note for extra details.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/buddyread announce <book> <audience> [note]
```
- `<book>` – The title of the book you want to read with a buddy
- `<audience>` – Who can see or respond to your announcement (e.g., "All Ages" or "Mature")
- `[note]` – (Optional) Add any extra details, preferences, or context for your future reading buddy

:::info
This command is great for letting others know you are looking to read a book with someone and share progress together.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/buddyread announce "Legends & Lattes" All ages Looking for a cozy, low-pressure reading partner!
```
This will:
- Create a public announcement in your globally, that you’re seeking a reading buddy for “Legends & Lattes”
- Display your note in the announcement, helping potential buddies know your preferences
- Add your announcement to the list that others can search or join

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Only one active buddy read announcement per book per user is allowed
> - Audiences are all ages, teen/ya, adult and mature
> - You can delete your announcement anytime with `/buddyread delete <id>`
> A well-written note can help you find a more compatible buddy!

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/buddyread announcements` – View all buddy read announcements

`/buddyread pair` – Pair up with a reading buddy from the announcement board

`/buddyread delete` – Delete one of your buddy read announcements

`/buddyread search` – Search announcements by book title
