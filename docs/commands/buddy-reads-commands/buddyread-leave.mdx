# /buddyread leave 

Leave an active buddy reading session if you no longer want to participate, feel uncomfortable with your buddy, or simply want to end the partnership.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/buddyread leave <book>
```
- `<book>` – The title of the book for the buddy read session you want to leave

:::info
When you use this command, both you and your buddy are notified and the original announcement is reposted to the global board for others to see and respond to.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/buddyread leave "Legends & Lattes"
```
This will:
- Remove you from the active buddy reading session for “Legends & Lattes”
- Notify both you and your buddy that the session has ended
- Automatically repost whichever user’s original announcement to the announcement board, so they can find a new buddy if they wish

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Use this command if you’re uncomfortable, your schedules don’t match, or you just want to opt out—no questions asked
> - The session is ended for both users, and the announcement goes back up to help your buddy find a new partner
> - Only use for active sessions; it will not affect finished or deleted sessions
> - Your previous DM history from the buddy read remains accessible

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/buddyread finish` – Mark a buddy reading session as finished

`/buddyread announce` – Post a new buddy read announcement

`/buddyread pair` – Pair up with a reading buddy

`/buddyread delete` – Delete your buddy read announcement
