# /buddyread status 

Show a summary of all your buddy reading sessions—active, finished, or even sessions you’ve left. Use this command to review your entire buddy read history and see stats or details for each session, all in one place.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/buddyread status [book]
```
- `[book]` – (Optional) Filter your buddy reading data to a specific book. If omitted, shows data for all sessions.

:::info
Remember that anything in `[ ]` is an optional parameter.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/buddyread status
```
This will:
- Display an overview of every buddy reading session you’ve ever participated in, including current, finished, and left sessions
- Show details such as book title, your buddy’s username, status (active, finished, left), start and end dates, and other session stats
```bash
/buddyread status "Legends & Lattes"
```
> This will show the status and details for just your “Legends & Lattes” buddy read session

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - This command is the best way to get a complete history of your buddy reading activity
> - Useful for reflecting on finished sessions or checking the status of any session, past or present
> - Stats may include total sessions, time spent, and partner info
> - Use `[book]` if you want to see details for just one session

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/buddyread sessions` – View only your currently active sessions

`/buddyread finish` – Mark a session as finished

`/buddyread leave` – Leave a buddy reading session

`/buddyread dm` – Message your buddy directly
