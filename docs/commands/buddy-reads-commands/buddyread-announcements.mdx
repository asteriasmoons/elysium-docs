# /buddyread announcements

View all current buddy read announcements across any server. This command lets you browse a global list of users who are seeking reading buddies, complete with book details, audience filters, and optional notes.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/buddyread announcements
```

:::info
No options or inputs are required — just use the command to display the latest announcements.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/buddyread announcements
```
This will:
- Display an embed or list of all active buddy read announcements from users in any server where Elysium Bot is present
- Show details for each announcement, including book title, audience filter (e.g., “All ages”, “Teen/YA”, “Adult”, or “Mature”), the user who posted, and any note
- Allow you to browse potential reading buddies by book or audience

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - All announcements are visible globally to anyone using Elysium, regardless of what server they’re in
> - The audience filter helps users find the right kind of reading buddy (content appropriateness, preferred age group, etc.)
> - You can pair with someone by using `/buddyread pair` and referencing their announcement
> - Announcements remain active until deleted or paired

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/buddyread announce` – Post your own buddy read announcement

`/buddyread pair` – Pair with a buddy from the announcements list

`/buddyread search` – Search for a buddy by book or audience

`/buddyread delete` – Delete your own buddy read announcement
