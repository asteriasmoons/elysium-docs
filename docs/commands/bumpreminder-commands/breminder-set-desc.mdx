# /bumpreminder set-description

Customize the body text of the bump reminder embed message that gets sent when your bump cooldown expires.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/bumpreminder set-description <description>
```
- `<description>` – The message that will appear inside the embed when the bump reminder is sent
:::warning
Sets the main content of the bump reminder message.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/bumpreminder set-description Time to bump the server! Use /bump now! 🚀
```
This will set the embed description to:
- Time to bump the server! Use `/bump` now! 🚀
- Help motivate users to bump with a clear and friendly message

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Markdown formatting (like `**bold**`, `*italics*`, etc.) is supported.
> - Keep it concise and inviting — this is the main message your community will see!
> - You can preview how this looks by running `/bumpreminder preview`.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/bumpreminder set-title` – Set the embed title text

`/bumpreminder preview` – View the full embed message

`/bumpreminder set-role` – Ping a role when sending the reminder

`/bumpreminder set-channel` – Choose where the reminder appears
