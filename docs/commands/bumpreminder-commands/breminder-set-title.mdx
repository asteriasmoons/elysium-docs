# /bumpreminder set-title

Set the title of the bump reminder embed message to make it stand out in your server.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/bumpreminder set-title <title>
```
- `<title>` – The title text that will appear at the top of the bump reminder embed
:::warning
Changes the bold title text at the top of the bump reminder embed.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/bumpreminder set-title It's Bump Time! 🚀
```
This will:
- Display a bold embed title that says: It's Bump Time! 🚀
- Help catch users' attention when the bump reminder is posted

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Keep titles short and eye-catching — this is what users will see first.
> - You can pair this with emojis or formatting to make it pop.
> - Use `/bumpreminder preview` to make sure your title and description look great together.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/bumpreminder set-description` – Customize the message content

`/bumpreminder preview` – See how your reminder currently looks

`/bumpreminder set-role` – Ping a specific role when the reminder is sent

`/bumpreminder set-channel` – Set where the reminder should be posted
