# /bumpreminder set-role

Choose a role to ping whenever the bump reminder message is sent. This helps notify specific members when it’s time to bump.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/bumpreminder set-role <role>
```
- `<role>` – The role that will be mentioned when the bump reminder is sent
:::warning
Adds a role mention to send with the bump reminder embed for better visibility.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/bumpreminder set-role @Bumpers
```
This will:
- Add @Bumpers as a ping above the bump reminder embed
- Make sure the right people are notified every time it’s time to bump

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Only one role can be assigned at a time.
> - Make sure the bot has permission to mention the selected role.
> - Great for teams or volunteers assigned to bumping the server regularly.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/bumpreminder set-description` – Set the message body

`/bumpreminder preview` – Preview the reminder with the role mention included

`/bumpreminder set-title` – Customize the embed title

`/bumpreminder set-channel` – Choose where the reminder message is posted
