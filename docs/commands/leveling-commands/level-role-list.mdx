# /level role-list

View a list of all level-based roles that have been configured on your server. This shows which roles are granted at which level milestones.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/level role-list
```
:::danger
No additional input required — just run it to view all active level-role assignments.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/level role-list
```
This will:
- Return a clean list of every configured level milestone
- Show the level number and the corresponding role that will be assigned
- Help you keep track of your role progression system

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - If no level-role pairs exist, Pandoryx will let you know the list is empty.
> - Use this to audit your leveling setup and ensure everything’s configured as expected.
> - You can remove or modify milestones using `/level role-remove` or `/level role-add.`

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/level role-add` – Create a new level milestone role

`/level role-remove` – Remove a level milestone

`/level set-announcement` – Customize level-up messages
