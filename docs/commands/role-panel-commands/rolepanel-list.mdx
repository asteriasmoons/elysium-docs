# /rolepanel list

List all the role panels you’ve configured in your server. Use this to quickly reference available panels or double-check their names before editing or publishing.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/rolepanel list
```
:::success
No input required — this command returns a list of all your current role panels.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/rolepanel list
```
This will:
- Display the names of all saved role panels
- Show the type (button or select) and possibly the number of roles added
- Help you confirm which panels are ready to be published or edited

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - This list is only for role panels created with `/rolepanel create.`
> - You can use this list to copy panel names for other commands like `/rolepanel publish`, `/delete`, or `/editembed.`
> - If no panels exist, Pandoryx will let you know.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/rolepanel create` – Set up a new panel

`/rolepanel delete` – Remove old or unused panels

`/rolepanel publish` – Post a panel to your server
