# /profile edit

Edit one or more fields in your Elysium book profile. Use this command to update your bio, intention, affirmation, current read, reading goal, favorite genre, preferred format, favorite author, or giftlist URL—without needing to fill in the entire profile.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/profile edit [bio] [intention] [affirmation] [current read] [reading goal] [favorite genre] [preferred format] [favorite author] [giftlist url]
```
Each field is optional—include only the fields you want to update.
- `[bio]` – A short description about you
- `[intention]` – Your current intention or focus
- `[affirmation]` – A positive statement for motivation
- `[current read]` – The book you’re currently reading
- `[reading goal]` – Your reading target (e.g., books/year)
- `[favorite genre]` – Your favorite book genre(s)
- `[preferred format]` – Hardcover, paperback, ebook, audio, etc.
- `[favorite author]` – Your favorite author(s)
- `[giftlist url]` – A link to your online giftlist or wishlist

:::success
**This command is good for editing parts of your profile instead of editing all of it.**
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/profile edit bio Bookworm, mood tracker, and coffee lover! current read Legends & Lattes
```
This will:
- Update your profile’s bio and current read fields
- Leave all other fields unchanged
- Instantly save your changes for everyone to see with `/profile view`

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - You can update as many or as few fields as you like in a single command
> - Great for making quick changes without re-entering your whole profile
> - Leaving a field out will keep its current value
> - Your profile is public to anyone using `/profile view`, but only you can edit your own info

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/profile set` – Set up your profile with all fields at once

`/profile view` – View yours or another user’s profile
