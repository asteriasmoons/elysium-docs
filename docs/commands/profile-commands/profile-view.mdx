# /profile view 

View your own or another user’s Elysium profile. Use this command to see a detailed summary of someone’s bookish interests, current read, reading goal, giftlist, and more—all in a tidy embed.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/profile view [user]
```
- `[user]` – (Optional) Mention another user to view their profile. If omitted, shows your own profile.

:::success
Great for getting to know community members, finding reading buddies, or sharing your 
profile with friends!

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/profile view
```
> This will display your current profile in an embed, showing all filled fields
```bash
/profile view @Jordan
```
> This will display @Jordan’s profile with their bio, favorite genre, current read, and other details

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - You can view anyone’s profile as long as they have set one up
> - If a user hasn’t filled out their profile, the embed will show empty or default fields
> - Your own profile is always visible to you and anyone who runs this command with your username
> - Perfect for sharing your book tastes, connecting over favorite authors, or trading giftlists

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/profile set` – Set up your complete profile

`/profile edit` – Edit specific profile fields
