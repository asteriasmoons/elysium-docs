# /embed list

View all embeds that have been created and saved on your server. Useful for managing your embed collection and remembering which names you’ve used.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/embed list
```

:::success
No additional input is required. This will return a full list of all saved embeds.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/embed list
```
This will:
- Display the names of all saved embeds
- Help you confirm names before using `/edit`, `/send`, or `/delete`
- Return the total count of embeds saved on your server

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Embed names must be unique per server.
> - If no embeds have been created yet, Pandoryx will let you know the list is empty.
> - Great for organizing content like welcome messages, rules, announcements, and more.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/embed create` – Create a new embed

`/embed edit` – Modify an existing embed

`/embed delete` – Remove an embed you no longer need
