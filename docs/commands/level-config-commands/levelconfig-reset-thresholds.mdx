# /levelconfig reset-thresholds

Reset your server’s custom level thresholds back to the default leveling structure. This clears any manual adjustments you’ve made to the level progression pace.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/levelconfig reset-thresholds
```
:::secondary
This command restores the baseline level thresholds used by Pandoryx.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/levelconfig reset-thresholds
```
This will:
- Remove all custom threshold values you've set with `/levelconfig set-thresholds`
- Revert your leveling curve to the default logic

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - This action affects how fast users level up — especially if you had a custom curve in place.
> - You can reconfigure custom thresholds again afterward using `/levelconfig set-thresholds.`

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/levelconfig view-thresholds` – See current threshold values

`/levelconfig set-thresholds` – Define a custom level curve
