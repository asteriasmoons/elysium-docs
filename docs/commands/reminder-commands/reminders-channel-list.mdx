# /reminders channel list 

List all designated channels for reading reminders in your server. Use this command to see where Elysium is allowed to send repeating reminders, so you can easily manage or update your notification setup.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/reminders channel list
```

:::warning
No options or inputs are required—just run the command to display all reminder channels.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/reminders channel list
```
This will:
- Display an embed with a list of all channels currently set to receive reading reminders
- Show each channel by name/mention so you can quickly reference or update them as needed

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Only channels added with `/reminders channel add` will appear here
> - Helpful for admins to review where reminders are going, especially in larger servers
> - Use this command to double-check before removing or adding more channels
> - If no channels are listed, reminders may only be going to DMs or have not been set up

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/reminders channel add` – Add a new channel for reminders

`/reminders channel remove` – Remove a designated reminder channel

`/reminders set` – Set a new reminder

`/reminders list` – View all your reminders
