# /reminder create

Create a new recurring reminder using an interactive panel. Reminders can be set to repeat on specific days and times, delivering messages or embeds to a selected channel.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/reminder create <name>
```
- `<name>` – A short name to identify the reminder (used for editing or deleting later).
:::secondary
This command opens a full configuration panel where you can customize the schedule, message content, channel, and more.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/reminder create hydration
```
This will:
- Open the reminder creation panel for a reminder called hydration
- Allow you to select the days, time, time zone, message text or embed, and delivery channel

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Reminder names must be unique across your server.
> - Use the `/reminder edit` command to adjust your settings later.
> - The bot must have permission to send messages in the selected channel for reminders to work.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/reminder edit` – Adjust a reminder after it’s created

`/reminder list` – View all configured reminders

`/reminder delete` – Remove a reminder from the schedule
