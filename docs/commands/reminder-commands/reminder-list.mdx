# /reminder list

View a full list of all reminders currently configured in your server. This gives you a quick snapshot of what’s scheduled and active.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/reminder list
```
:::secondary
No additional options needed — just run the command to see all your reminders.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/reminder list
```
This will:
- Display the names of all active reminders
- Show their scheduled times and the channels they’ll post in
- Help you manage or edit reminders without guessing their names

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - If no reminders are configured, Pandoryx will let you know.
> - This list includes recurring reminders only — one-time reminders are not currently supported.
> - Use the names shown here with `/reminder edit` or `/reminder delete.`

<hr className="md-divider-gradient" />

🔍 Related Commands
`/reminder create` – Set up a new reminder

`/reminder delete` – Remove an old reminder

`/reminder edit` – Update timing or content
