# /autorole view

Display a list of all roles currently set to be automatically assigned when new users join the server.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/autorole view
```
:::secondary
No additional input is required — this command simply shows your configured autoroles.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/autorole view
```
This will:
- Return a list of all roles that are set to be auto-assigned
- Help you confirm which roles are active in the autorole system

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - If no autoroles are set, Pandoryx will let you know the list is empty.
> - Combine with `/autorole add` or `/autorole remove` to adjust your setup as needed.
> - Autoroles are applied automatically when someone joins — no further action needed.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/autorole add` – Add a role to the autorole system

`/autorole remove` – Remove a role from the autorole system
