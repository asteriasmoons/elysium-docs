# /report 

Send feedback, a bug report, or a suggestion directly to the developer of Pandoryx. Use this to communicate issues or share improvement ideas.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/report <message>
```
- `<message>` – A short and clear description of the issue, bug, or suggestion
:::secondary
This is for general reports — not for critical bot errors (which should be posted in the Bug Reports Issue).
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/report The /warn view command isn't showing anything in DMs.
```
This will:
- Send your message directly to the developer behind the scenes
- Include your username and server for reference
- Help improve the bot over time

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Use this for minor bugs, feedback, or feature ideas.
> - For serious bugs, visit the Bug Reports Issue and comment there instead.
> - Abuse of this command may result in it being disabled for your server.
