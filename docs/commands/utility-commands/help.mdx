# /help

Open a clean, paginated embed menu that displays all available Pandoryx commands with brief descriptions. Great for quick reference and navigation.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/help
```
:::secondary
Displays a series of embeds containing categorized command lists with short descriptions. Navigation is handled with buttons below the embed.
:::

<hr className="md-divider-gradient" />

### 💜 Features
- 🔘 Clickable navigation buttons – Use ⏮️ Prev and ⏭️ Next to flip through command categories
- 🧩 Each page is a separate embed – Commands are grouped and shown as clean, static embeds with short descriptions
- ⛔ Next button greys out on the last page – You'll know when you've reached the end of the help pages

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/help
```
This will:
- Show an embed like Moderation Commands with entries
- Let you tap Next to go to the next page (e.g., Embed Tools, Utility, etc.)
- Disable the Next button once you hit the final page

<hr className="md-divider-gradient" />
