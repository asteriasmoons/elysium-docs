# /journal edit 

Edit a specific journal entry by its number. Use this command to update or correct an entry in your private journal—perfect for making changes, adding more detail, or fixing mistakes.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/journal edit <index or number>
```
- `<index or number>` – The number of the journal entry you want to edit (as shown in your journal list)

:::warning
**After running the command, you’ll be prompted with a modal to enter your updated text for that entry.**
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/journal edit 2
```
This will:
- Prompt you to enter new or updated text for your second journal entry
- Save your changes to the selected entry in your private journal
- Confirm the edit with a message or embed

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - You can only edit entries in your own journal
> - Use `/journal list` to review entry numbers and details before editing
> - Entry numbers remain consistent—editing does not change their order or timestamps
> - Useful for clarifying thoughts, fixing typos, or expanding on earlier reflections

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/journal list` – See all your journal entries

`/journal view` – View a specific entry in full

`/journal add` – Add a new journal entry

`/journal delete` – Remove an entry from your journal
