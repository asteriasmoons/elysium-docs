# /journal view 

View an entire journal entry by its number. Use this command to read the full text of any entry in your private journal—perfect for reflection, copying, or sharing with someone you trust.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/journal view <index or number>
```
- `<index or number>` – The number of the journal entry you want to view (as listed in /journal list)

:::warning
**The `<index/number>` is not the same as `[page]`**
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/journal view 5
```
This will:
- Display the full text of your fifth journal entry in an embed or message
- Allow you to read the complete entry, including everything you wrote when adding or editing it

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Only your own entries can be viewed with this command—your journal remains private
> - Use `/journal list` to find entry numbers before viewing
> - Great for reflecting on past entries, finding inspiration, or revisiting old thoughts
> - You can copy or use the text elsewhere as needed

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/journal list` – Browse your journal entries and find their numbers

`/journal edit` – Edit a journal entry

`/journal add` – Add new entries

`/journal delete` – Remove an entry
