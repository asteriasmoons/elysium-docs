# /progress leaderboard 

See everyone’s reading progress streaks and who’s on top! This command displays a leaderboard of users in your server, ranked by how many consecutive days they’ve logged their reading progress.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/progress leaderboard
```

:::success
**No options or inputs are required—just run the command to view the leaderboard.**
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/progress leaderboard
```
This will:
- Show an embed listing users and their current reading streaks (number of consecutive days logged)
- Rank users from highest to lowest streak, so you can see who’s maintaining the most consistent reading habit
- Motivate everyone to keep their streak going!

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - The leaderboard only includes users from your current server
> - Only progress logged with /progress log counts toward streaks
> - Streaks reset if a day is missed
> - A fun way to build friendly competition and encourage daily reading

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/progress streak` – View your own reading streak

`/progress log` – Log your daily reading progress

`/progress history` – View your reading progress history
