# /afk disable

Manually remove your AFK status. Use this when you’re back and want to stop showing your AFK message to others.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/afk disable
```
:::success
Turns off AFK mode for your user and clears your stored message.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/afk disable
```
This will:
- Remove your AFK status
- Prevent the bot from showing an away message when you're mentioned or replied to

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - You can also be removed from AFK automatically if your server has message-based auto-clear enabled (see `/afk nomessage`).
> - This is the cleanest way to exit AFK mode if auto-clear is turned off.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/afk enable` – Set an AFK message that appears when you're pinged

`/afk nomessage` – Toggle auto-clear when sending a message
