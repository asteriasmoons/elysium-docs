# /sticky embed list

Display all sticky embeds that are currently saved on your server. Use this to review, manage, or choose which embed to send or edit.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/sticky embed list
```
:::success
No additional input required. This will return a list of all stored sticky embeds.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/sticky embed list
```
This will:
- Show the names of all saved sticky embeds
- Help you confirm which ones are available to send, edit, or delete
- Provide a reference for using other sticky embed commands

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Only sticky embeds created with `/sticky embed create` will appear here.
> - This list may also show which embeds are currently active in channels (depending on configuration).
> - If no sticky embeds are found, Pandoryx will notify you that the list is empty.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/sticky embed create` – Create a new sticky embed

`/sticky embed edit` – Update embed content

`/sticky embed send` – Post a sticky embed into a channel
