# /sticky embed delete

Delete a saved sticky embed from your server’s storage. This will permanently remove it from the list of available sticky embeds.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/sticky embed delete <name>
```
- `<name>` – The name of the sticky embed you want to delete. Must match an existing saved sticky embed.

:::success
This action is permanent — it deletes the embed from your saved list and disables its use going forward.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/sticky embed delete rules
```
This will:
- Remove the sticky embed named rules from your server’s saved sticky embeds
- Stop it from being used or posted in any channel

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Deleting a sticky embed does not automatically remove it from a channel. If it's still active, use `/sticky embed remove` to take it down from a channel manually.
> - Use `/sticky embed list` if you’re unsure which embeds are saved.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/sticky embed create` – Make a new sticky embed

`/sticky embed list` – View your saved sticky embeds

`/sticky embed remove` – Remove a sticky embed from a specific channel
