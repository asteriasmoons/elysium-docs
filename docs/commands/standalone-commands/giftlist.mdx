# /giftlist 

Send an announcement that you gifted a user with a book from their giftlist that you found with `/profile view [user]`. This command helps celebrate generosity and lets everyone know about book gifting within your community.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/giftlist <@user> <book title> <author>
```
- `<@user>` – The Discord user you are announcing the gift for
- `<book title>` – The title of the book you gifted
- `<author>` – The author of the book

:::warning
Great command for announcing who you have gifted a book to!
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/giftlist @Jordan "Legends & Lattes" "Travis Baldree"
```
This will:
- Announce in the designated channel that you’ve gifted “Legends & Lattes” by Travis Baldree to @Jordan
- Encourage celebration of book gifting within your community

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Always mention the user to ensure they’re notified
> - Make sure the book and author match what’s on the user’s giftlist (see `/profile view [user]`)
> - Use quotation marks for multi-word titles and author names for clarity
> - This command is for celebratory announcements only; the bot does not handle actual transactions
> - Perfect for book clubs, gifting events, or just spreading positivity!

<hr className="md-divider-gradient" />
