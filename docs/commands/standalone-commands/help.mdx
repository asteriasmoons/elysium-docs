# /help 

Look up information about the bot and the commands the bot has as well as the features and functionality of the bot.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/help
```
:::warning
No options are required. This command works as-is.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/help
```
This will:
- Display an embed with Elysium’s main features and command categories
- Offer tips for getting started or navigating the bot’s functionality
- May include links or buttons to explore further

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Works in any channel or DM where the bot has permission
> - Ideal for first-time users or those exploring new features
> - Acts as a central reference point for what Elysium can do

<hr className="md-divider-gradient" />

### 🔍 Related Commands
None — but this command connects you to everything else.
