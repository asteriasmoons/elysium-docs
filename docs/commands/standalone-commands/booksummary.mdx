# /booksummary

Get a summary of any book you’re considering reading. Use this command to quickly receive a concise, spoiler-free overview of a book’s plot, themes, and general vibes—perfect for deciding if a title is right for you or refreshing your memory before diving in!

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/booksummary <title> [author]
```
- `<title>` – The title of the book you want a summary for
- `[author]` – (Optional) The author of the book, to help narrow down results (especially for books with common titles)

:::warning
If you use this command, it is better if you include the author in the case of the book not being one of a kind.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/booksummary "Legends & Lattes"
```
This will:
> Send you a clear, brief summary of “Legends & Lattes,” highlighting the main story, mood, and themes—without major spoilers
```bash
/booksummary "The Hobbit" Tolkien
```
This will:
> Return a summary for “The Hobbit” by J.R.R. Tolkien, making sure you get the right edition/author if there are duplicates

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Summaries are concise and aim to be spoiler-free
> - Works for most published books—fiction, nonfiction, classics, and more
> - Adding the author is helpful for books with common titles
> - Perfect for previewing books, picking your next read, or remembering details before book club
> - If the bot can’t find a summary, it will let you know and suggest checking your title/author spelling
