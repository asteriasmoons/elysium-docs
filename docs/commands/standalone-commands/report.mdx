# /report 

Send feedback or bug reports to the developer.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/report <message>
```
- `<message>` – The issue, suggestion, or feedback you want to send. Be clear and specific!

:::warning
This command allows you to directly send reports and feedback to the dev.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/report The /journal edit command isn’t saving changes when used in DMs.
```
This will:
- Privately send your report to the bot developer
- Help improve the bot by sharing real user experiences
- Allow the dev to investigate bugs or consider your feature requests

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Reports are private and do not appear in chat or logs visible to others
> - You can use this for suggestions, errors, weird behavior, or general feedback
> - The more detailed your report, the more helpful it is!
