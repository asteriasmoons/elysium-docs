# /autoreact remove

Disable auto-reactions in a specific channel. Messages sent there will no longer receive automatic emoji reactions.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/autoreact remove <channel>
```
- `<channel>` – The text channel you want to remove from the auto-reaction system
:::success
Stops auto-reacting to messages in the specified channel.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/autoreact remove #suggestions
```
This will:
- Stop the bot from adding any automatic emoji reactions in `#suggestions`
- Free up the channel from system-added reactions

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - You can use `/autoreact list` first to see which channels are active.
> - This command will only work if the specified channel was previously configured with auto-reactions.
> - Useful for reverting testing or cleaning up emoji usage.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/autoreact set` – Enable auto-reactions in a channel

`/autoreact edit` – Change emojis for a currently active channel

`/autoreact list` – View your current auto-reaction configurations
