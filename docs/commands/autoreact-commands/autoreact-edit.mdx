# /autoreact edit

Edit the emojis that are automatically added as reactions to every message in a specified channel.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/autoreact edit <channel> <emoji1> <emoji2> [emoji3] [emoji4] [emoji5] [emoji6]
```
- `<channel>` – The text channel you want to update
- `<emoji1>` – The first emoji that will be used for auto-reacting
- `<emoji2>` – The second emoji that will be used for auto-reacting
- `[emoji3]` – (Optional) The third emoji that will be used for auto reacting.
- `[emoji4]` – (Optional) The fourth emoji that will be used for auto reacting.
- `[emoji5]` – (Optional) The fifth emoji that will be used for auto reacting.
- `[emoji6]` – (Optional) The sixth emoji that will be used for auto reacting.
:::success
Updates the emojis used for auto-reaction in a specific channel.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/autoreact edit #suggestions 👍 👎
```
This will:
- Change the auto-reactions in `#suggestions` to 👍 and 👎
- Replace any previously configured emojis for that channel

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Custom emojis are supported.
> - Both the first two emojis are required for this command.
> - The bot must have permission to use and react with the selected emojis in the specified channel.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/autoreact set` – Enable auto-reactions in a new channel

`/autoreact remove` – Remove auto-reactions from a channel

`/autoreact list` – View all auto-reaction channels and settings
