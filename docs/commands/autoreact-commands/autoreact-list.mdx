# /autoreact list

View all channels in your server that have auto-reactions enabled, along with the emojis assigned to each.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/autoreact list
```
:::success
Displays a list of all configured auto-reaction channels and their assigned emojis.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/autoreact list
```
This will:
- Send an embed showing all channels currently set for auto-reaction
- Show the emojis configured for each listed channel
- Help you manage or clean up auto-react setups

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - This command is view-only — to make changes, use `/autoreact edit` or `/autoreact remove`.
> - A quick way to double-check emoji setups across multiple channels.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/autoreact set` – Add auto-reactions to a channel

`/autoreact edit` – Change the emojis for a configured channel

`/autoreact remove` – Disable auto-reactions in a channel
