# /tarot

Draw a tarot spread for insight, guidance, or self-reflection. Use this command to receive either a one-card or three-card tarot reading—perfect for daily inspiration, quick clarity, or a deeper dive into your current situation.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/tarot <one/three>
```
- `<one/three>` – Choose one for a single card pull or three for a classic three-card spread

:::secondary
The tarot command is great for getting insight on a situation or intention you have.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/tarot one
```
> This will pull one tarot card and display its name, image, and a brief interpretation

```bash
/tarot three
```
This will:
- Pull three tarot cards, each with its own position (typically Past, Present, Future or Situation, Action, Outcome)
- Show each card’s name, image, and interpretation in order

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Tarot readings are random and generated fresh every time
> - Each card comes with an accessible, encouraging interpretation—no doom and gloom
> - Great for journaling prompts, self-discovery, or daily guidance
> - You can use the command as often as you’d like for new readings
