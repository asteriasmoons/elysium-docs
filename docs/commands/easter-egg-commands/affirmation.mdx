# /affirmation

Get a supportive, uplifting affirmation. Use this command to receive a gentle boost of encouragement or a positive thought—perfect for daily self-care, journaling, or whenever you need a kind word.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/affirmation
```
:::secondary
No options or inputs are required—just run the command to get your affirmation.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/affirmation
```
This will:
- Instantly send you a randomly chosen affirmation from Elysium’s curated list
- Provide a gentle, empowering statement to brighten your day or guide your intentions
- Great for sharing with friends, using in rituals, or starting your morning mindfully

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Affirmations are selected from a diverse, positive collection
> - You can use this command as often as you like—get as many affirmations as you need
> - Wonderful as a journaling prompt, meditation focus, or daily mantra
> - If you want more, just run the command again for a new affirmation
