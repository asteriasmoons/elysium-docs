# /auto forum-list

View a list of all forum channels in your server that have automatic messages enabled.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/auto forum-list
```
:::warning
Shows all forums currently using the auto forum panel system.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/auto forum-list
```
This will:
- Display an embed listing all forums where auto forums are active
- Help admins quickly see which forums are set up for automated message creation

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Use this command to audit or manage your server’s forum-based ticket system.
> - You can combine this with `/auto forum-remove` to clean up unused or outdated forums.
> Only forums added with `/auto forum-add` will appear in this list.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/auto forum-add` – Enable auto messages in a new forum

`/auto forum-remove` – Remove a forum from the auto forum system

`/auto forum-set` – Configure or customize the message sent in a forum post
