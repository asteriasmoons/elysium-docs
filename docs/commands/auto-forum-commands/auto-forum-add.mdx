# /auto forum-add

Add a forum channel to your auto forum messages system. This allows Pandoryx to automatically post a message panel with a button in that forum for users to open new threads with the right format.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/auto forum-add <forum>
```
- `<forum>` – The forum channel where you want automatic message panels to be sent
:::warning
Registers a forum as part of the server’s auto forum system.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/auto forum-add #support-forum
```
This will:
- Add `#support-forum` to the list of forums monitored by Pandoryx
- Enable the bot to post message panels or intro messages automatically in that forum
:::warning
**This command posts a message you configure in the posts created in a forum with a button. Use them however you'd like but they're especially great for forum ticketing.**
:::

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Only forum channels can be added with this command.
> - The bot needs permission to view and send messages in the forum.
> - You can customize the ticket panel/message per forum with `/auto forum-set`.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/auto forum-list` – View all forums with auto forum panels enabled

`/auto forum-remove` – Remove a forum from the auto forum system

`/auto forum-set` – Customize the auto message, embed, or panel in the forum
