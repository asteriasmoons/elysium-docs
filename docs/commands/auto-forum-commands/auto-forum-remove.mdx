# /auto forum-remove 

Remove a forum from your auto forum system. The bot will no longer post automated starter messages in that forum.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/auto forum-remove <forum>
```
- `<forum>` – The forum channel you want to remove from the auto forum system
:::warning
Disables auto messages for a specified forum channel.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/auto forum-remove #support-forum
```
This will:
- Remove `#support-forum` from the bot’s list of active auto forums
- Stop any further automated messages from being posted there

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - This does not delete the forum or its threads — it just removes the auto message functionality.
> - You can always re-add the forum later with `/auto forum-add`.
> - Good for cleaning up unused or no-longer-needed auto forum setups.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/auto forum-add` – Add a forum to the auto ticket system

`/auto forum-list` – View all forums with auto ticket panels enabled

`/auto forum-set` – Customize the auto panel/message per forum
