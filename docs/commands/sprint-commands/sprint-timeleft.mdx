# /sprint timeleft 

See how much time is left in the current reading sprint. Use this command to quickly check the remaining minutes and seconds—so you can pace yourself, plan your last pages, or know when to get ready for the finish!

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/sprint timeleft
```

:::warning
No options or inputs are required—just run the command any time during a sprint.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/sprint timeleft
```
This will:
- Show the amount of time remaining in the active sprint (e.g., “14 minutes, 20 seconds left”)
- Let you and other participants stay aware of the countdown
- Help you make the most of your sprint time

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Only works if a sprint is currently active in your server
> - Anyone can use this command during a sprint
> - Useful for timing breaks, sprints, or wrapping up your reading session
> - Once the timer runs out, use `/sprint finish` to log your ending page

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/sprint join` – Join the current sprint

`/sprint finish` – Submit your ending page

`/sprint leaderboard` – See who read the most

`/sprint start` – Start a new sprint

`/sprint end` – End the sprint manually (admin only)
