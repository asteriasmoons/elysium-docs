# /sprint leaderboard 

View the reading sprint leaderboard for your server. Use this command to see who read the most pages during the latest sprint—and celebrate the top readers!

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/sprint leaderboard
```

:::warning
No options or inputs are required—just run the command to see the results.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/sprint leaderboard
```
This will:
- Display an embed showing the participants in the most recent (or active) sprint
- Rank everyone by the number of pages read
- Highlight the top reader(s) and everyone’s results for motivation and bragging rights

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Leaderboards are server-specific and reflect the most recent or active sprint
> - Only users who joined and submitted starting/ending pages are included
> - Use this for friendly competition and to inspire more reading in your community
> - A new leaderboard is generated for each sprint

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/sprint start` – Begin a new reading sprint

`/sprint join` – Join a sprint with your starting page

`/sprint finish` – Submit your ending page

`/sprint end` – End a sprint manually (admin only)

`/sprint timeleft` – See how much time remains
