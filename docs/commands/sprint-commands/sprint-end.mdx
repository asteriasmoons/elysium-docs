# /sprint end 

Manually end a reading sprint before the scheduled finish. This command is for admins only and is useful for wrapping up a sprint early due to scheduling conflicts or special circumstances.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/sprint end
```

:::warning
No options or inputs are required—just run the command to immediately end the active sprint.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/sprint end
```
This will:
- Instantly end the current active reading sprint
- Stop the timer and prompt participants to submit their ending page numbers
- Trigger final announcements and update the sprint leaderboard

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Only users with admin permissions can run this command
> - If no sprint is active, the command will let you know
> - Useful if the sprint needs to be ended early for any reason

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/sprint start` – Begin a new reading sprint

`/sprint timeleft` – Check how much time remains in an active sprint

`/sprint finish` – Submit your ending page number

`/sprint leaderboard` – View the server’s sprint leaderboard
