# /starboard enable 

Turn on the starboard system for your server so users can highlight messages with a star emoji (or custom emoji).

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/starboard enable
```
:::warning
Activates the starboard feature.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/starboard enable
```
This will:
- Activate the starboard feature in your server
- Begin tracking starred messages and preparing to repost them once they meet the threshold

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Make sure you’ve also set a starboard channel using `/starboard set-channel`.
> - The system will use the current emoji and threshold settings. Adjust those if needed.
> - Only messages that reach the threshold of reactions will be posted.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/starboard set-channel` – Choose where starred messages are posted

`/starboard set-emoji` – Pick the emoji users should react with

`/starboard set-threshold` – Set how many stars a message needs

`/starboard disable` – Turn the starboard off
