# /starboard set-threshold

Set the number of emoji reactions required on a message before it gets posted to the starboard.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/starboard set-threshold <number>
```
- `<number>` – The minimum number of emoji reactions needed to send a message to the starboard
:::warning
Controls how many stars a message must receive before being highlighted.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/starboard set-threshold 3
```
:::warning
This will require at least 3 reactions using the configured emoji before a message is posted in the starboard channel
:::

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Only reactions with the set emoji (from `/starboard set-emoji`) count toward the threshold.
> - Setting a low number (like 1 or 2) might result in more frequent posts — choose a value that fits your server size and vibe.
> - You can adjust this at any time without needing to re-enable the feature.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/starboard enable` – Start the starboard system

`/starboard set-emoji` – Define which emoji triggers a “star”

`/starboard set-channel` – Pick the destination for starred posts

`/starboard disable` – Turn the starboard system off
