# /starboard disable

Turn off the starboard system in your server, stopping all tracking and posting of starred messages.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/starboard disable
```
:::warning
Disables the starboard feature and prevents any new starred messages from being posted.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/starboard disable
```
This will:
- Immediately deactivate the starboard system
- Stop any new messages from being posted to the configured starboard channel

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Previously starred messages will remain in the starboard channel unless removed manually.
> - You can re-enable the system at any time using `/starboard enable`.
> - No configuration (channel, emoji, threshold) will be lost by disabling.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/starboard enable` – Re-activate the starboard

`/starboard set-channel` – Assign where starred messages go

`/starboard set-emoji` – Pick the reaction emoji

`/starboard set-threshold` – Adjust how many stars a message needs
