# /autodelete remove 

Remove a channel from your autodelete configuration, stopping it from automatically deleting messages.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/autodelete remove <channel>
```
- `<channel>` – The text channel you want to remove from the autodelete system
:::danger
Disables autodeletion for a specific channel.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/autodelete remove #spam
```
This will:
- Turn off message autodeletion in `#spam`
- Allow messages to remain in the channel as normal

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - You can use `/autodelete list` first to see which channels are currently active.
> - This command will not affect channels that were never configured for autodelete.
> - Only users with appropriate permissions can remove autodelete settings.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/autodelete set` – Enable autodeletion for a channel

`/autodelete list` – View all active autodelete configurations
