# /ticketpanel edit

Edit the appearance or configuration of an existing ticket panel using an interactive menu. This lets you update the panel’s message, embed, or setup without recreating it.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/ticketpanel edit <name>
```
- `<name>` – The name of the ticket panel you want to edit. Must match an existing panel.

:::success
Only ticket panels that have already been created with /ticketpanel create can be edited.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/ticketpanel edit support
```
This will:
- Open an interactive editor for the support ticket panel to edit
- Button emoji
- Embed title, description, or color
- Confirmation messages shown to users

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - The editor is UI-based and easy to navigate — no need to remember parameters.
> - This command does not repost the panel — use `/ticketpanel post` to update the live version after editing.
> - Great for tweaking existing panels without starting over.

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/ticketpanel create` – Start a new ticket panel

`/ticketpanel post` – Post or repost a panel to a channel

`/ticketpanel delete` – Remove a panel completely
