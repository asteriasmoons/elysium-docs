# /recommend language 

Set your preferred language for book recommendations using a standard language code. Use this command to make sure Elysium suggests books in your preferred reading language.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/recommend language <language>
```
- `<language>` – The language code you want to receive recommendations in (e.g., EN for English, ES for Spanish, FR for French, etc.).

:::secondary
**This should support any language that you speak.**
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/recommend language EN
```
This will:
- Set your recommendation language to English
- Future book recs will prioritize books in the EN (English) language

<hr className="md-divider-gradient" />

### 🗃️ Notes
**Use standard language codes—some examples:**
> - EN (English), ES (Spanish), FR (French), DE (German), IT (Italian), etc.
> - You can update your language code at any time
> - The more widely used the language, the more recommendations you’ll get
> - This setting only affects your personal recommendations

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/recommend genres` – Set your favorite genres

`/recommend set` – Set all your recommendation preferences at once

`/recommend get` – Get a book recommendation instantly

`/recommend notify` – Choose where your recs are sent

`/recommend show` – View your rec settings
