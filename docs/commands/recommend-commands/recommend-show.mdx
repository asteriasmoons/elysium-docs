# /recommend show

Show your current book recommendations and preferences. Use this command to see a summary of your personalized rec settings—including your chosen genres, language, delivery location, and frequency—as well as any active book recommendations you’ve received.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/recommend show
```

:::secondary
**No options or inputs are required—just run the command to display your settings and current recs.**
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/recommend show
```
This will:
- Display an embed showing your current recommendation preferences (genres, language code, frequency, and where recs are delivered)
- List any active book recommendations you haven’t dismissed yet, including book titles, authors, and links
- Help you review and adjust your settings as needed

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Great for double-checking your setup before updating with `/recommend set`
> - All displayed info is private—only you can see your recs and settings
> - If you’re not receiving recs, make sure your notification location is set and your DMs are open
> - You can update any preference at any time with the related commands

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/recommend set` – Update all your rec preferences at once

`/recommend genres` – Set your preferred genres

`/recommend interval` – Choose how often you get recs

`/recommend notify` – Set where recs are sent

`/recommend language` – Set your preferred language code

`/recommend get` – Instantly get a new recommendation
