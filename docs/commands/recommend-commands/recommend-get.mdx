# /recommend get 

Get a new book recommendation immediately. Use this command whenever you want a fresh book rec based on your set preferences—perfect for finding your next great read on demand!

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/recommend get
```

:::secondary
No options or inputs are required—just run the command for an instant recommendation.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/recommend get
```
This will:
- Instantly provide you with a book recommendation based on your saved genres, language, and preferences
- Show details like the book’s title, author, genre, and a link (such as Goodreads)
- Make it easy to discover new books whenever you’re in the mood

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Recommendations are personalized based on your previously set preferences
> - If you haven’t set any preferences, you’ll get a random book suggestion
> - Run this command as often as you like for more recs
> - Great for breaking reading slumps or trying something new

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/recommend genres` – Set your favorite genres

`/recommend set` – Set all preferences at once

`/recommend show` – View your current rec preferences

`/recommend interval` – Set how often you receive automatic recs
