# /recommend set 

Set all your book recommendation preferences at once. Use this command to quickly update your frequency, genres, notification location, and language code in a single step—perfect for setting up or refreshing your Elysium book rec experience!

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/recommend set <interval> <genres> <notify> <language>
```
- `<interval>` – How often you want to receive automatic recommendations (daily, weekly, monthly, or none)
- `<genres>` – Your favorite genres (separate multiple with commas or spaces)
- `<notify>` – Where to send your recs (DMs for private messages, or mention a channel, e.g., #reading-corner)
- `<language>` – The language code for recommendations (e.g., EN, ES, FR)

:::secondary
**You can use this command if you prefer not to use command after command to set things up.**
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/recommend set weekly Fantasy, Mystery DMs EN
```
This will:
- Schedule book recommendations every week
- Filter recs to the Fantasy and Mystery genres
- Send your recs privately to your DMs
- Recommend books in English (EN)

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - All fields are required to use this command
> - This is the fastest way to set up all your preferences in one go
> - You can update your settings at any time with this command or the individual commands
> - Make sure Elysium has permission to DM you or to post in the specified channel

<hr className="md-divider-gradient" />
