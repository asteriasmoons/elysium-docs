# /review book 

View all reviews for a specific book. Use this command to see ratings and feedback left by the Elysium community for a particular title—perfect for deciding what to read next or comparing opinions.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/review book <book>
```
- `<book>` – The title of the book whose reviews you want to see

:::info
**Please report any nasty reviews you may see to the support server.**
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/review book "Legends & Lattes"
```
This will:
- Show an embed listing all reviews for “Legends & Lattes”
- Include reviewer names, star ratings, review text, and the author for each entry
- Help you get a sense of community opinions and enthusiasm for the book

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Reviews are global—all Elysium users’ reviews for this book are included
> - Each entry shows who wrote the review and their rating
> - If there are no reviews for the book, you’ll get a friendly message instead
> - Use this to help decide if a book is worth picking up or recommending to others

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/review author` – View reviews by a specific author

`/review add` – Add your own review

`/review list` – Browse all reviews

`/review view` – Shortcut to view reviews for a book

`/review edit` – Edit your existing review

`/review delete` – Delete your review
