# /review list 

List all reviews in Elysium’s global review library. Use this command to browse every book review submitted by the community, making it easy to discover new titles, see trending books, or explore recent feedback from other readers.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/review list
```

:::info
No options or inputs are required—just run the command to view the global review list.
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/review list
```
This will:
- Display an embed or paginated list of all reviews in Elysium
- Show each review’s book title, author, reviewer, rating, and a preview of the review text
- Let you browse and discover books through the lens of the community’s feedback

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Reviews are listed globally—anyone using Elysium can see and add to the collection
> - If there are many reviews, results may be paginated; use navigation to see more
> - Use other review commands to filter by author, book, or to view/edit your own reviews
> - A great way to find popular reads or hidden gems

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/review add` – Submit your own review

`/review book` – Filter reviews by book

`/review author` – Filter reviews by author

`/review view` – Shortcut to view reviews for a specific book

`/review edit` – Update your review

`/review delete` – Remove your review
