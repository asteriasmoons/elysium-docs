# /review view 

Quickly view all reviews for a specific book. This shortcut command instantly displays community ratings and feedback for a chosen title—helpful for deciding what to read next or checking what others thought.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/review view <book>
```
- `<book>` – The title of the book you want to see reviews for

:::info
**This command is basically an alias for `/review book`**
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/review view "Legends & Lattes"
```
This will:
- Show an embed with all reviews for “Legends & Lattes”
- Display reviewer names, their star ratings, the review text, and the book’s author
- Make it easy to get an overview of the book’s reputation in the Elysium community

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - This is a shortcut to `/review book`—they work the same way
> - Reviews are global, from all Elysium users
> - If no reviews are found, you’ll get a friendly message instead
> - Use related commands to add, edit, or filter reviews further

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/review book` – View reviews for a book (same as this command)

`/review author` – Browse reviews by author

`/review add` – Submit your own review

`/review edit` – Edit a review you wrote

`/review delete` – Remove your review

`/review list` – Browse all global reviews
