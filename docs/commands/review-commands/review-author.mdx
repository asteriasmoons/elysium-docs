# /review author 

View all reviews for books by a specific author. Use this command to browse community feedback and ratings for any author’s works, making it easier to find your next great read or see what others thought.

<hr className="md-divider-gradient" />

### 💬 Usage
```bash
/review author <author>
```
- `<author>` – The name of the author whose book reviews you want to see

:::info
**So this command is great for when you want author reviews only.**
:::

<hr className="md-divider-gradient" />

### 📌 Example
```bash
/review author Travis Baldree
```
This will:
- Display an embed listing all reviews for books written by Travis Baldree
- Show ratings, review texts, and book titles for each entry
- Help you spot highly recommended titles or trending authors in the community

<hr className="md-divider-gradient" />

### 🗃️ Notes
> - Reviews are collected globally—results include reviews from all Elysium users
> - Each review includes the reviewer’s name, the book title, star rating, and their written feedback
> - If no reviews exist for the author, you’ll see a friendly message letting you know
> - Use this to discover hidden gems or get a sense of an author’s style

<hr className="md-divider-gradient" />

### 🔍 Related Commands
`/review book` – View reviews by book title

`/review add` – Write your own review

`/review list` – Browse all reviews

`/review view` – Shortcut to view reviews for a book

`/review edit` – Update your review

`/review delete` – Remove your review
